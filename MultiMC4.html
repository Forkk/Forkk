MultiMC 4 Manage Minecraft installations with ease!

What is MultiMC?

MultiMC is a custom launcher that makes managing Minecraft easier by allowing you to have multiple installations of Minecraft at once. This means that you can have two different .minecraft folders each with it's own mods saves texture packs, etc. without having to waste your time fooling around with .bat files. It also allows you to easily install and remove mods without you even having to touch WinRAR or 7zip at all. This allows you to spend less time trying to get your mods to work and more time actually enjoying Minecraft.

How do I install it?

Windows
  1. Download MultiMC from the link below.
  2. Make a folder somewhere for it.
  3. Place the file you downloaded in the folder.
  4. Double click the file.
  5. Enjoy using MultiMC!
Mac OS X
  1. Download MultiMC from the link below.
  2. Extract the zip file.
  3. Place it wherever you want and double click it to start it.
  4. Enjoy using MultiMC!
Linux
  1. Download MultiMC from the link below.
  2. Make a folder somewhere for it.
  3. Place the file you downloaded in the folder.
  4. Make sure you set the file to be executable. On Ubuntu, this is usually in the permissions tab on in the file properties dialog.
  5. Double click the file.
  6. Enjoy using MultiMC!

Features

  • Easily manage multiple installations of Minecraft at once.
  • Supports multiple different versions of Minecraft at once.
  • Works with every version of Minecraft since alpha 1.0.11!
  • Install and remove mods by simply dragging and dropping them or copying and pasting them into a list.
  • Easily downgrade to older versions of Minecraft using MultiMC's built-in MCNostalgia!
  • Start Minecraft with a custom window size (or maximized)!
  • Easily change Java's memory allocations in the settings.
  • Shows Minecraft's console output in a convenient color coded window.
  • Allows you to easily kill Minecraft if it freezes.
  • Shows custom icon and window title on the Minecraft window.
  • And much more!
