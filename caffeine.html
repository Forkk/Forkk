Caffeine Keep your computer awake.

What is Caffeine?

Caffeine is a program that keeps your computer from falling asleep. This can be used to keep you screensaver from running when you're listening to music or whatever you're doing.

How do I install it?

Caffeine is very easy to install. Simply make sure you have the latest version of the .NET framework, download Caffeine from the link below, and run it.
